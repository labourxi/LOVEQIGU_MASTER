# 🌿 AR游伴小程序视觉规范系统 V1.0

---

## 1. System Identity

AR游伴 is a lightweight AR-based cultural exploration mini program.

It is NOT a game.

It is NOT a reward system.

It is a real-world cultural discovery interface.

---

## 2. Visual Positioning

Core visual language:

- Eastern aesthetics
- Minimal UI
- Low saturation color system
- Soft lighting
- Paper / scroll / ink texture feel
- Subtle AR overlay

---

## 3. Forbidden Visual Styles

The following styles are strictly forbidden:

- Game UI (levels, XP, rewards, loot)
- High saturation neon cyberpunk
- Competitive ranking systems
- Gacha / card draw visuals
- Heavy sci-fi HUD interfaces

---

## 4. Color System

### Primary Colors
- Ink Black
- Deep Night Blue
- Mist Gray

### Accent Colors
- Soft Gold
- Jade Green (low saturation)
- Warm White Light

### Forbidden Colors
- Neon green
- Neon purple
- High saturation red

---

## 5. Typography System

- Primary Chinese font: Serif-style Songti (优先思源宋体)
- English font: clean sans-serif with low visual weight
- Avoid bold aggressive typography

---

## 6. Animation Language

All animations must follow:

- Fade in
- Soft float
- Breathing glow
- Gentle expansion
- Slow dissolve

NO fast motion, NO aggressive transitions.

---

## 7. UI Structure Rules

### 7.1 Home Page
- Minimal exploration entry
- Soft recommendation cards
- Cultural atmosphere background
- Low information density

---

### 7.2 Location Detail Page
- Scroll-based expansion layout
- Ink scroll reveal animation
- Cultural narrative focus
- AR entry point as secondary action

---

### 7.3 AR Exploration Layer
- Real camera is primary
- AR elements must be minimal
- Use micro-glow indicators only
- No heavy overlays

---

### 7.4 Artifact (Relic) System
Artifacts represent cultural memory fragments.

Rules:
- No rarity labels (NO SSR/UR)
- No numeric stats
- No gamification mechanics
- Only narrative + origin + meaning

---

## 8. Interaction Design Principles

- Discovery over reward
- Meaning over score
- Experience over accumulation

---

## 9. System Principle

AR游伴 is a cultural perception layer that connects real-world locations with subtle AR-based meaning discovery.
