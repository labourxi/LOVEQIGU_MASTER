# AR游伴 · 城市合伙人模型

**文件 ID：** `04_business/city_partner_model.md`  
**版本：** V1 初稿  

---

## 1. 概念定义

城市合伙人 = 在单一城市内复制 **空间操作系统 + 内容工厂**，非加盟商卖牌。

---

## 2. 结构说明

### 2.1 三阶段

| 阶段 | 范围 | 交付 |
|------|------|------|
| P1 | 1景区3点 | 试点 SOP 验证 |
| P2 | 1城市N景区 | 标准化部署包 |
| P3 | 城际信物图谱 | 文化 IP 网络（需 Canon 审批） |

### 2.2 角色

| 角色 | 职责 |
|------|------|
| 平台 | OS · 审查 · 工厂 |
| 城市合伙人 | 本地 BD · 景区关系 · 运营 |
| 景区 | 现场 · 内容确认 |
| 商家 | 权益供给 |

---

## 3. 流程说明

```text
试点成功 → 城市独家/排他谈判 → 授权费 + 分成（合同定义）
         → 景区批量导入模板 → 统一数据语法
```

---

## 4. 示例

```text
某市 2027 目标：3 景区 · 9 探索点 · 30 商家
复制物：pilot_deployment_sop + runtime package 模板
```

---

## 5. 可执行说明

- 新城市首单仍走 90 天 MVU  
- 城际图谱叙事不得自创 Canon  
- 授权合同模板（法务待备）
