status: APPROVED

## 核心原则

AR游伴后台不是普通 Admin 系统。

后台属于：

```text
AR游伴东方接福体系
```

因此：

```text
平台后台
园区后台
商家后台
培训中心
审核中心
```

必须统一使用福文化视觉语言。

## 一级颜色体系

### Blessing Red

用途：

```text
主按钮
发布活动
审核通过
重要运营动作
```

建议：

```css
#C83A30
```

### Blessing Gold

用途：

```text
福印
荣誉状态
高级运营指标
```

建议：

```css
#B68A3D
```

### Ink Black

用途：

```text
正文
标题
表格
```

建议：

```css
#2F2A26
```

### Paper White

用途：

```text
背景
页面容器
卡片
```

建议：

```css
#F8F6F2
```

## 云纹体系

允许：

```text
Header
Banner
活动页顶部
```

透明度：

```text
5%-8%
```

禁止：

```text
大面积古风贴图
```

## 福印体系

替代：

```text
奖章
成就图标
```

统一使用：

```text
福印
```

场景：

```text
培训完成
审核通过
活动完成
商家认证
```

## 后台命名规范

- Merchant Center -> 福礼商家中心
- Coupon Center -> 福礼中心
- Activity Center -> 接福活动中心
- Reward Center -> 福印中心
- Collection Center -> 福册中心

## 设计目标

实现：

```text
前台与后台同一文化体系
统一东方视觉语言
统一福文化心智
```

避免：

```text
前台东方文化
后台标准SaaS
```
