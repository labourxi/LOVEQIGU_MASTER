# ADMIN_COMPONENT_LIBRARY_V1

## Objective

建设后台统一组件库，供以下三类后台复用：

- Platform Admin
- Merchant Portal
- Park Admin

## Component List

- TopNav
- SideNav
- Breadcrumb
- PageHeader
- KPI Card
- Table
- Filter
- Status Badge
- Button
- Input
- Select
- Textarea
- Modal
- Drawer
- Pagination
- Empty State
- Loading State
- Error State

## Design Language

参考 `BACKOFFICE_DESIGN_SYSTEM_V1`，统一使用：

- 赭石
- 纸白
- 墨色

## Output

- `apps/admin/shared/components/`

## Constraints

- 不接 API
- 不接数据库
- 不接 Runtime

## Validation

- 组件可独立渲染
