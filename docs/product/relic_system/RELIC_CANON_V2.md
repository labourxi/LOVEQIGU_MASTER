# Token Definition

## 信物

定义：

被找回的连接

---

说明：

信物不是物件。

信物不是装备。

信物是一段被重新找回的连接。

* 连接恢复记录
* 图谱点亮
* 用户成长轨迹

功能。

定义：

# Lighting Definition

定义：

连接恢复

失落连接被重新找回。

看见

这段连接被找回。

---

神门穴被点亮

表示：

安定

这段连接被找回。
