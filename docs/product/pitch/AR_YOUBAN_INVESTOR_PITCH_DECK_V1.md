# AR游伴 · 商业路演 PPT V1

**文档类型：** 投资人路演幻灯片脚本（20–25 页）  
**产品品牌：** AR游伴（ARYOUBAN）  
**Slogan：** 看见即是找回  
**视觉基调：** 东方空间美学 · 星图 / 经纬线 / 墨色 · 极简 · 高级感 · 非游戏化  
**术语约束：** 探索地图 · 信物 · 权益中心 · 信物 ≠ 数字藏品  

> 使用说明：每页含 **标题 / 图示结构 / 页脚结论**。图示为设计执行说明，可直接交付视觉设计师制版。

---

## 全局视觉规范（Design System）

| 维度 | 规范 |
|------|------|
| 主色 | 墨绿 `#263A34` · 宣纸 `#F4F1EB` · 暖金点缀 `#B68A3D` |
| 辅色 | 淡墨灰 `#6B7280` · 经纬线 `#C9B896`（15% 透明度） |
| 字体 | 标题：思源宋体 / 霞鹜文楷；正文：思源黑体 Light |
| 图形语言 | 细线星图、经络经纬、圆点节点、留白 ≥ 40% |
| 禁止 | 霓虹 HUD、宝箱/等级/战力、抽卡动效、游戏 UI 边框 |

**页脚统一格式：** `结论 · 一句话`（右下，12pt，淡墨）
