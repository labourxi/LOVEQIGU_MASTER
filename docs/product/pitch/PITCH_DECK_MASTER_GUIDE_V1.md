## 母版视觉规范

### 色彩

| 角色 | 色值 | 用途 |
|------|------|------|
| 墨绿 Ink | `#263A34` | 封面/强调块/表头 |
| 宣纸 Paper | `#F4F1EB` | 内容页背景 |
| 暖金 Gold | `#B68A3D` | 标题下划线、节点、高亮列 |

| 经纬线 Line | `#C9B896` | 星链、分隔线、装饰 |

### 版式元素（每页固定）

1. **顶栏金线** — 6px 高度暖色条
2. **右上星链** — 3 圆点 + 细线（品牌识别）
3. **标题区** — 28pt 加粗 + 金色短下划线
4. **页脚结论** — 右下 11pt：「结论 · …」
5. **左下品牌** — `AR游伴 · ARYOUBAN`

## 编辑建议

### 在 PowerPoint 中

1. 打开 `AR_YOUBAN_INVESTOR_PITCH_DECK_V1.pptx`
2. **视图 → 备注** 查看演讲者备注（关键页已预埋）
3. 融资页（Slide 23）填入实际 **金额 / 估值 / 轮次**
4. 封底（Slide 25）填入 **联系人 / 邮箱 / 微信二维码**
5. 可插入产品截图至 Slide 09–10 旁白页（建议新增附录页）

### 批量再生成

```bash
python scripts/pitch/generate_investor_pitch_deck.py
```

修改 `scripts/pitch/generate_investor_pitch_deck.py` 中 `build_slides()` 的文案后执行。

### 字体替换（可选升级）

当前默认：**微软雅黑**（Windows 兼容）

设计升级建议：
- 标题：思源宋体 / 霞鹜文楷
- 正文：思源黑体 Light

在 PPT「设计 → 字体」中全局替换即可，母版颜色无需改动。
