## 1. 概念定义

## 2. 结构说明

## 3. 流程说明

路演 Q&A 处理：

```text
问题 → 对照本表 / 技术边界 → 事实回答 → 承诺跟进项写 issue
```

## 4. 示例

| 问题 | 回答要点 |
|------|----------|
| 是游戏吗？ | 否。无等级抽卡；信物=故事进度 |
| 信物=NFT？ | 否。信物≠数字藏品 |
| AR 挂了？ | 备用显现同一闭环；UI/XR 解耦 |
| 怎么赚钱？ | 景区 SaaS+内容+商家；C 端免费 |
| 护城河？ | 空间 OS+Canon+可交付试点 |
| 第一个客户？ | 1景区3点90天 MVU |
| 真 AR 就绪？ | REAL_AR_READY: NO；Marker 阻塞；产品闭环 PASS |
| 市场规模数字？ | 万亿级框架；具体需公开报告核验 |
| 融资用途？ | 40%产品 30%市场 20%内容 10%合规 |
