## §1 文件说明

本文件用于 **登记** [`LOVEQIGU_WORLD_RULES_CANON_V1.md`](world/LOVEQIGU_WORLD_RULES_CANON_V1.md) 中：

- 尚未定义的区域  
- 部分定义、禁止越界扩展的区域  
- 明确禁止推断的区域  
- 待未来 Canon 版本补充的区域  

**定位**：介于 **L0 正典** 与 **L1 World Bible** 之间的 **边界控制文件**。

```text
LOVEQIGU_WORLD_RULES_CANON_V1（L0 · 已冻结）
        ↓
CANON_GAP_REGISTRY（L0.5 · 空白登记 · 本文件）
        ↓
LOVEQIGU_WORLD_BIBLE_V1（L1 · 仅转述 Canon 已述部分）
```

编写 **WORLD_BIBLE**、**CHAPTER**、**Design**、**Narrative** 前 **必须先查阅本文件**。

| 本文件 | 不是 |
|--------|------|
| 世界观正文 | 设定集 |
| 空白填充稿 | 创作素材 |
| Canon 补丁 | 隐式立法 |

**未在本 Registry 中解除冻结（或 Canon 未升级）的 Gap，不得补写。**

### 硬性约束

1. **不得** 修改 `LOVEQIGU_WORLD_RULES_CANON_V1`  
2. **不得** 新增任何世界规则  
3. **不得** 补充 Canon 未明确说明的内容（如创世神、宇宙起源细节、高维文明、前文明、神明体系、超自然组织等）  
4. **仅登记状态**，不负责填充内容

## §2 状态分类

| 状态 | 代码 | 含义 | 编写权限 |
|------|------|------|----------|
| **Canon Defined** | `DEFINED` | 已在 Canon 中明确说明 | 允许引用、转述 |
| **Canon Partial** | `PARTIAL` | 已有部分定义 | 仅可在 Canon 已述范围内引用；**禁止**扩展超出 Canon 的部分 |
| **Canon Undeclared** | `UNDECLARED` | Canon 未定义 | **不得脑补**；等待未来 Canon 升级 |
| **Canon Forbidden** | `FORBIDDEN` | 明确禁止推导 | 即使推测合理 **也不得** 写入任何层级文档 |

## §4 使用规则

### 4.1 禁止填补 Gap

以下 **不得** 用 Gap 登记项作为借口自行定义世界观：

| 文档 / 产出 | 规则 |
|-------------|------|
| **WORLD_BIBLE** | **不得** 填补 Gap；仅转述 `DEFINED` / 允许范围内的 `PARTIAL` |
| **CH01～CH100** | **不得** 定义 Gap；章节事件不得引入 `UNDECLARED` 前提 |
| **秒哒页面文案** | **不得** 定义 Gap |
| **Design System** | **不得** 定义 Gap |
| **Legacy 文档**（CREATION_MEMO 等） | 冲突时 **L0 胜**；Legacy 不得填补 Gap |

### 4.2 允许行为

- 引用 Canon 原文或经批准的转述  
- 引用状态为 `DEFINED` 的 Registry 项  
- 引用状态为 `PARTIAL` 的 Registry 项中 **Canon 已明确** 的部分  
- 引用 Canon 已述 **历史事件**（大分化、遗忘、探索开始、归真纪元）  
- 在 L2/L4 使用 **产品配置**（印谱位序、积分上限），须标注 `[L2]` / `[L4]`，且不升格为 Canon  

### 4.3 编写前检查清单

```text
1. 查 Registry：目标主题状态是否为 UNDECLARED / FORBIDDEN？
2. 若是 → 停止扩写；改用 Canon 已述表述或留空
3. 若是 PARTIAL → 核对 Canon 依据列，不得越界
4. 若是 DEFINED → 可转述，不得改写本质
5. 新发现空白 → 登记本 Registry，不先写 Bible/章节
```

## §5 Gap 解除流程

任何 Gap 内容的 **正式补充** 必须走 Canon 升级流程，**不得** 通过 Bible 或章节「先行填补」。

```text
1. 创建变更提案
   └── docs/world/proposals/CANON_CHANGE_PROPOSAL_{YYYYMMDD}_{slug}.md

2. 经创世层 Canon 审核
   └── 见 LOVEQIGU_CANON_INDEX §1

3. 升级 Canon
   └── 发布新版本（例：LOVEQIGU_WORLD_RULES_CANON_V1.1.md）

4. 更新本 Registry
   └── 将对应项状态改为 DEFINED 或 PARTIAL（并填写 Canon 依据）

5. 方可写入 WORLD_BIBLE
   └── L1 仅转述新 Canon 已述部分
```

**跳过步骤 1–4 直接在 Bible / CHAPTER 中补写 Gap，视为违规扩写。**
