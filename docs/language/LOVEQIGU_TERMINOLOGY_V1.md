## 0. 使用说明

### 0.1 字段定义

| 列 | 含义 |
|----|------|
| **文案ID** | 唯一标识 · 格式 `T-{域}-{序号}` |
| **原文 (old)** | 当前仓库用户可见文案 |
| **新文案 (new)** | 宪法合规目标文案 |
| **层级** | `L1` 商业 · `L2` 产品 · `L3` 世界观 · `过渡` 按 §6 分场景 |

### 0.2 实施规则

1. **old = new** 表示已合规，实施时可跳过。  
2. **拆分** 条目需在同一 commit 内同步删除混合句。  
3. **L3 仪式弹窗** 不得保留 `心愿值 +N` · `成就` · `禁用词` · `积分`。  
4. 工程路由名 / 字段名 **不在本表**（仅用户可见 copy）。
