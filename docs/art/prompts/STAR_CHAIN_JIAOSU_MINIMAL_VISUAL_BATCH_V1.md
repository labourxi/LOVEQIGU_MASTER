# STAR_CHAIN_JIAOSU_MINIMAL_VISUAL_BATCH_V1

## Task Name

STAR_CHAIN_JIAOSU_MINIMAL_VISUAL_BATCH_V1

## Asset Category

STAR_CHAIN

## Use Scene

信物档案页 / 星链档案 / 角宿最小视觉单元

## Visual Goal

设计"角宿两星一线"的最小星链视觉样张，用于验证星链系统的基础视觉方向。

该样张应表现：

* 两个星位
* 一条细线连接
* 其中一个节点已点亮
* 另一个节点保持暗态
* 整体呈东方古星图 / 古籍天文图谱气质

## Must Include

* 两个圆形星点节点
* 一条非常克制的细线连接
* 一个点亮节点
* 一个未点亮节点
* 节点周围柔和光晕
* 古籍纸本或暗青灰夜空背景
* 可作为前端节点状态参考

## Must Avoid

* 五角星
* 现代科幻 HUD
* 霓虹蓝紫
* 亮金爆炸
* 金属徽章
* 抽卡稀有奖励感
* 战力盘
* 复杂星云背景
* 过度装饰

## Style Keywords

东方星图，古籍天文图谱，克制留白，低饱和，旧金，暗青，柔白星光，微粒，纸本质感，观测线

## Output Ratio

建议 1:1 或 4:3

## Transparent Background

NO

## Need Layering For Motion

YES

## Master Prompt

设计一张东方古星图风格的"角宿两星一线"最小视觉样张。画面中只有两个星位节点，一条细而克制的观测线连接它们。其中一个节点柔和点亮，另一个节点保持暗淡。整体像古籍天文图谱或宣纸上的星宿记录，低饱和，留白，暗青灰与旧金色调，柔白微光，轻微颗粒质感。星点必须是圆点、微孔或星孔，不能是五角星。不要科幻HUD，不要霓虹，不要亮金爆炸，不要金属徽章，不要抽卡感。画面应适合后续前端拆成节点、连线、光晕、标签等元素。
