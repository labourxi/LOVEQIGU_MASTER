# MERIDIAN_NODE_MINIMAL_VISUAL_BATCH_V1

## Task Name

MERIDIAN_NODE_MINIMAL_VISUAL_BATCH_V1

## Asset Category

MERIDIAN

## Use Scene

信物档案页 / 经络档案 / 经络多穴位最小视觉单元

## Visual Goal

设计一条经络线与多个穴位节点的最小视觉样张，用于验证经络系统的基础视觉方向。

## Must Include

* 一条柔和经络路径线
* 多个穴位节点
* 一个节点处于点亮态
* 其他节点保持暗态
* 局部流光或气脉贯通感
* 古法经脉图气质

## Must Avoid

* 医学教材风
* 真实人体器官
* 红蓝血管图
* 霓虹医疗 UI
* 荧光绿
* 强科技扫描
* 电流冲击感

## Style Keywords

古法经脉图，淡墨线条，宣纸，朱砂暗红，温金，柔白节点光，气脉流动，低饱和，东方身体观

## Output Ratio

建议 1:1 或 4:3

## Transparent Background

NO

## Need Layering For Motion

YES
