## Visual Goal

设计"莲花印"宝物的实体视觉方向。

注意：

用户打卡掉落的是"宝物"，不是"信物"。

莲花印是具体宝物本体；它可以绑定信物别名"角宿一（室女座 α，Spica）"，但画面不要直接把角宿星图画成宝物。

## Must Include

* 一个实体化莲花印
* 古朴材质
* 东方收藏物气质
* 可作为 2D 插画或未来 3D 模型参考
* 不要过度金属徽章化

## Must Avoid

* 金属徽章贴面
* 亮金爆炸
* 抽卡稀有奖励感
* 现代游戏装备
* 过度奢华珠宝
* 直接做成星座图标
* 五角星

## Style Keywords

古朴莲纹，印章，玉石，旧铜，陶土，朱砂痕，东方收藏物，低饱和，克制，纸本展示，文物感
