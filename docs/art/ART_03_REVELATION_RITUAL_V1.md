# 一、文档目标

- 星图显现
- 遗珍显现
- 宝物显现
- 回响显现

所有后续动画、Lottie、AR特效必须遵守本规范。

---

# 二、核心哲学

显现 ≠ 出现

显现 ≠ 获得

显现 ≠ 生成

显现 = 遮蔽消散

用户并非获得新的事物。

用户只是重新看见原本存在的事物。

# 五、视觉原则

原则1：

原本存在

所有显现对象必须被理解为：

一直存在。

只是此前未被看见。

---

原则2：

缓慢浮现

禁止突然出现。

禁止瞬间生成。

---

原则3：

关系优先

优先显现：

- 刻线
- 路径
- 星痕
- 关联

而非物体本身。

---

原则4：

留白

显现结束后保留静默时间。

允许用户感受回响。

---

# 六、动态原则

允许：

- 尘落
- 雾散
- 纹理浮现
- 微光流动
- 回响扩散

禁止：

- 爆闪
- 掉落
- 召唤
- 爆炸
- 飞行
- RPG特效
