# 🌍 3. HOME 2 — EARTH INTERFACE

## 3.1 Core Concept

Earth is a global visualization of:

- user nodes
- regional resonance distribution
- content density
- consciousness hotspots

---

## 3.2 Interaction Model

Users can:

- rotate globe freely
- zoom into continents → countries → cities
- click glowing nodes
- search for specific locations or users

---

## 3.3 Lighting System

Earth brightness depends on regional resonance:

- High resonance → warm golden light
- Medium resonance → soft blue light
- Low resonance → dim cold tones
- inactive regions → minimal visibility

---

## 3.4 Node System

Each node represents:

- a user
- a content fragment
- a memory entry
- or a consciousness trace

Node types:

- Small Node (normal users)
- Medium Node (active contributors)
- Large Node (high resonance creators)

---

## 3.5 Node Entry Rules

Nodes may be accessed if:

- public permission granted
- resonance threshold met
- search match confirmed
- invitation access enabled

---

## 3.6 Search System

Supports:

### Basic Search
- user name
- location
- node title

### Advanced Filters
- content type
- resonance level
- region
- activity level

---

## 3.7 Earth Purpose

Earth View is not a map.

It is a:

- consciousness distribution interface
- human resonance topology map
- global experiential navigation layer
