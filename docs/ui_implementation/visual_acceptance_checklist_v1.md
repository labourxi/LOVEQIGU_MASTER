# visual_acceptance_checklist_v1

## Landing 验收
1. 首屏是否足够克制
2. 是否避免工具化
3. 是否有单一焦点
4. 是否具备世界感
5. CTA 是否唯一明确
6. 是否保留“留在足迹里 / 收藏世界”

## Explore Home 验收
1. 是否不像后台或工具软件
2. 是否避免表格感
3. 是否具备沉浸式探索卡片流
4. 是否避免奖励化
5. 是否避免数值化
6. 搜索是否轻量
7. 卡片是否具有城市 / 记忆 / 线索氛围

## 总体验收
1. 是否不新增世界观内容
2. 是否不新增系统
3. 是否不新增页面
4. 是否可直接交付前端还原
5. 是否保持东方克制风格
