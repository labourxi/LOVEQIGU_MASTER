**布局规则：**

- 单卡，圆角 16rpx，墨底 `#263A34` 8% 透明度或纸底描边  
- 角标「推荐前行」用 `--gold` 小标签  
- 「前往探索」为 ghost 按钮，不得与 Hero CTA 同尺寸

## 8. 滚动与首屏约束

| 区块 | 首屏是否必须可见 |
|------|------------------|
| Hero 全量 | ✅ 必须 |
| 今日推荐卡片 | ✅ 至少露出标题 + 卡片上 40% |
| 探索进度 | 建议露出标题 |
| 最近获得 | 可滚动后可见 |
| 登录条 | 未登录时可滚动可见 |

**禁止首屏出现：** 第二 Primary 按钮、功能矩阵、权益/信物频道入口标题。
