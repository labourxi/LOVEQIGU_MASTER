# landing_page_final_v1

## 1. 页面定位
Landing Page 不是产品首页，而是进入探索世界的入口页。  
目标是在 3 秒内让用户进入“世界即将显现”的情绪状态。

## 2. 页面目标
关键词：
- 留白
- 仪式感
- 东方克制
- 轻探索
- 未被完全点亮的世界

## 3. 页面结构
自上而下结构如下。

### A. 顶部
- 极简 Logo / AR游伴
- 居中
- 无复杂导航
- 无多入口按钮

### B. 主视觉区
背景：
- 深墨黑到深靛蓝渐变
- 轻雾
- 稀疏呼吸粒子
- 极弱星点

中央主体：
- 一张隐约展开的地图 / 星图结构
- 不完整
- 不清晰
- 像未被完全点亮的世界
- 带轻微发光轮廓
- 带低强度层次深度

### C. 唯一主按钮
按钮文案：
进入探索

要求：
- 唯一主 CTA
- 半透明浅色
- 带轻微呼吸感
- 不高饱和
- 不强促销感
- 点击后进入 Explore Home

### D. 底部文案
底部居中，小字：
留在足迹里  
收藏世界

## 4. 视觉要求
- 页面必须克制
- 不工具化
- 不像后台
- 不像功能列表
- 不像电商
- 不像游戏大厅

## 5. 配色建议
- 背景主色：`#0B0F14`
- 主光：`#BFC9D6`
- 辅助灰蓝：`#7E8FA3`

## 6. 动效建议
只允许：
- 呼吸
- 渐隐
- 漂浮
- 雾化
- 微弱粒子移动

禁止：
- 弹跳
- 爆闪
- 奖励闪光
- 剧烈缩放
- 数值反馈动效

## 7. 响应式与前端还原重点
- 主视觉必须始终居中
- 首屏信息必须少
- 第一屏不允许出现复杂信息区块
- 按钮和地图主体必须形成明显焦点
- 保证移动端首屏完整性

## 8. 前端组件拆解
- LandingPage
- LogoBlock
- AmbientBackground
- GradientLayer
- ParticleLayer
- MistLayer
- CentralMapPreview
- PrimaryCTAButton
- FooterTagline
