**版本：** V1.0  
**语言层：** Hero 允许 L2+L3；其余区块 **仅 L2 产品层**

---

## 1. 文案原则

| 原则 | 执行 |
|------|------|
| 一句主行动 | 全页只有「开始探索」称主按钮 |
| 不促销 | 无积分、折扣、签到 |
| 不游戏化 | 无等级、战力、排行榜 |
| 场域感 | 景区名、探索点、信物名用实词 |
| 首版回归 | 「留在足迹里」「收藏世界」作副标语，不拆成两个功能名 |

| 元素 | 文案 | 层级 |

| 状态 | 文案 |
