# ⚙️ 2. FUNCTION SPEC

## 2.1 积分系统

STATE:
userRights.points

RULE:
- 每次 AR 打卡成功 → +1
- 每次获得信物 → +1 bonus
- 页面自动刷新显示

---

## 2.2 领取奖励

BUTTON: “领取”
ACTION:
- validate if available
- change status: available → claimed
- push to reward history

---

## 2.3 兑换权益

BUTTON: “兑换”
ACTION:
- check points balance
- deduct points
- generate coupon instance

STATE UPDATE:
userRights.points -= cost
userRights.coupons.push(newCoupon)

---

## 2.4 查看历史记录

BUTTON:
- reward feed item click

ACTION:
→ open detail modal

---

## 2.5 导航行为

- 查看信物 → /relic/index
- 返回探索 → /explore/index
- 我的 → /my/index
