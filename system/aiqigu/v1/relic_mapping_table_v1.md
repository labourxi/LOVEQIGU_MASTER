## 场景 → 信物 对照

### 1. 入口广场 → 入口印记

- **来源图片**：5张（Snipaste_*22-31-57*, *22-32-36*, *12-06-13* + 微信图片_*1_2*, *2_2*）
- **锚定方式**：扫描入口广场地面铺装图案
- **触发条件**：GPS定位在入口广场半径20m内
- **交互流程**：到达 → 扫描 → 印记显现 → 收集 → 探索开始
- **视觉阶段**：锚定 → 浮现（1200ms） → 显现（3次脉冲） → 确认

### 2. 古树区域 → 林荫印记

- **来源图片**：2张（Snipaste_*12-07-47*, 微信图片_*12_2*）
- **锚定方式**：扫描古树树干/树根自然纹理
- **触发条件**：必须已收集#1 + GPS定位在古树半径10m内
- **交互流程**：步行到古树 → 扫描树干 → 印记在树根浮现 → 光幕扩散至树冠 → 收集
- **视觉阶段**：锚定 → 浮现（1600ms） → 扩散（2000ms） → 确认

### 3. 江南老街 → 老街印记

- **来源图片**：4张（Snipaste_*12-06-45*, *12-06-59*, *12-07-18*, 微信图片_*12_2*）
- **锚定方式**：扫描店铺门头/招牌
- **触发条件**：必须已收集#2 + GPS定位在江南老街范围内
- **交互流程**：进入老街 → 扫描门头 → 金色刻线由一侧向另一侧延展 → 收集
- **视觉阶段**：锚定 → 浮现（1800ms） → 刻线（1000ms） → 确认

### 4. 爱企谷咖啡 → 咖啡印记

- **来源图片**：3张（微信图片_*13_2*, *14_2*, *16_2*）
- **锚定方式**：扫描桌面小卡/杯垫（首推）或Logo墙
- **触发条件**：必须已收集#3 + GPS定位在咖啡店室内
- **交互流程**：就座 → 看到桌面小卡 → 扫描 → 印记从中心扩散 → 金色光尘沉降 → 收集
- **视觉阶段**：锚定 → 浮现（1400ms） → 光尘（1200ms） → 确认

### 5. 爱企谷书店 → 书页印记

- **来源图片**：2张（微信图片_*15_2*, *17_2*）
- **锚定方式**：扫描书店门头（首推）或书籍封面
- **触发条件**：必须已收集#4 + GPS定位在书店室内
- **交互流程**：进入书店 → 扫描门头 → 印记如书页翻开 → 刻线如书法写下 → 合拢为印记
- **视觉阶段**：锚定 → 浮现（1800ms） → 刻线（1400ms） → 确认

### 6. 爱企谷手作馆 → 手作印记

- **来源图片**：4张（微信图片_*3_2*, *4_2*, *7_2*, *15_2*）
- **锚定方式**：扫描工作台面或完成的作品
- **触发条件**：必须已收集#5 + GPS定位在手作馆室内
- **交互流程**：完成手作 → 扫描作品/台面 → 印记在作品旁浮出 → 场波纹扩散 → 合影后收集
- **视觉阶段**：锚定 → 浮现（1600ms） → 场扩散（1200ms） → 确认

### 7. 中心广场 → 中心印记

- **来源图片**：4张（Snipaste_*22-36-36*, *22-38-53*, 微信图片_*10_2*, *11_2*）
- **锚定方式**：扫描中心广场地面特色铺装图案
- **触发条件**：必须已收集全部#1-#6 + GPS定位在中心广场半径25m内
- **交互流程**：到达中心广场 → 扫描地面 → 6枚印记光点汇入 → 汇聚为中心印记 → 完成
- **视觉阶段**：锚定 → 汇集（2800ms） → 融合（1200ms） → 成型（1600ms） → 确认
