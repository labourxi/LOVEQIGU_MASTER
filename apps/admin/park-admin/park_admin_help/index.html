快速指南

如何查看园区数据总览？
进入左侧「数据总览」，首屏可查看入驻商家、活动、探索点、用户参与与卡券发放核销等核心指标。
如何查看活动数据？
进入「活动数据」，在数据看板 Tab 查看各活动参与、发券与核销效果；活动管理 Tab 可轻量查看与跳转详情。
如何查看商家表现？
进入「商家数据」，查看各商家领取量、核销量与核销率，关注「表现较好」与「需关注」提示。
如何创建活动草稿？
进入「创建活动」，填写活动名称、时间、关联商家与探索点，保存草稿后可在活动详情继续查看。
如何查看发布检查结果？
进入「发布检查」或活动详情中的发布检查入口，查看检查结论详情、阻断原因与平台优化意见。

常见问题

为什么活动无法发布？
常见原因为礼遇配置尚未完成平台审核。请在「发布检查」查看阻断原因与平台意见，或提交工单咨询。
商家数据多久更新？
当前为 Mock 演示数据。正式环境将按日活动数据同步，具体频率以平台通知为准。
卡券核销率偏低怎么办？
查看数据总览与商家数据中的优化建议，协调商家完善到店引导，必要时联系平台协助优化探索路径。
如何联系平台处理发布阻断？
在发布检查页查看完整结论详情后，可通过「提交工单咨询」说明活动名称与阻断原因。
