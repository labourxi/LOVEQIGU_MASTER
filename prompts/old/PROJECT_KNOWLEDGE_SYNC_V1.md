# PROJECT_KNOWLEDGE_SYNC_V1

Objective

Synchronize newly added Canon / Bible / Governance files
into the project knowledge layer.

Purpose

Not only detect files.

Also extract:

- key rules
- frozen constraints
- terminology
- required dependencies
- governance impact

--------------------------------------------------
PART 1

Load latest inventory and registry.

--------------------------------------------------
PART 2

Find newly added or modified files matching:

* CANON
* BIBLE
* GOVERNANCE
* TERMINOLOGY
* HOMEPAGE
* EXPLORE_MAP
* RIGHTS
* BLESSING
* SCENIC

--------------------------------------------------
PART 3

Read content.

Generate summaries:

File Name

Purpose

Core Rules

Frozen Constraints

Dependencies

Affected Tasks

--------------------------------------------------
PART 4

Generate:

docs/PROJECT_KNOWLEDGE_SYNC_REPORT.md

--------------------------------------------------
PART 5

Output:

NEW KNOWLEDGE

UPDATED KNOWLEDGE

CONFLICTS

REGISTRY IMPACT

MEMORY IMPACT

Success Marker:

PROJECT_KNOWLEDGE_SYNC_COMPLETE = YES
