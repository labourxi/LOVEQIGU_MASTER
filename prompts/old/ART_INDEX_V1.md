# ART_INDEX_V1

Objective

Create an index file for the docs/art folder.

Target output:

docs/art/ART_INDEX_V1.md

Requirements:

1. Scan docs/art/

2. Register all existing art-related files.

3. For each file, record:

- file name
- relative path
- purpose
- status if detectable
- related task
- dependency relationship

4. Define art load order for future ART tasks.

5. At minimum, include sections:

- Visual Bible
- Star Activation Ritual
- Visual Asset Specs
- Relic / Revelation / Ritual Art
- Asset Package References
- Implementation Dependencies

6. Define conflict rules:

- ART_INDEX_V1 is an index only
- Do not modify existing ART canon files
- Do not rename files
- Do not move files
- Do not create new visual rules
- If duplicate or unclear files are found, list them as WARN

7. Output marker:

ART_INDEX_V1_COMPLETE = YES

Also generate:

docs/art/ART_INDEX_V1_REPORT.md
